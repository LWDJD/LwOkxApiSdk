## 简介

LWOkxApiSdk 是一个用于与 OKX API V5 交互的 Java SDK。它目前处于开发阶段，提供了一些基本的功能，但尚未稳定。

## 特点

- **轻量级**：只实现了 OKX API 的核心功能。
- **易用性**：简化了与 API 的交云，方便快速开发。
